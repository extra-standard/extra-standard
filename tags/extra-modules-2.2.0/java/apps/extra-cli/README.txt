Verzeichnisbeschreibung
--------------------------------
/bin
Hier stehen ausführbare Beispiels-Dateien zur Verfügung. Die Datei extra-cli.bat kann aus dem Windows-Console gestartet werden. 
Die Umsetzung in Linux ist analog zu der extra-cli.bat. Zu den jeweiligen Fachverfahren werden spezielle Auslieferungsdateien
mitgeliefert.

/lib
Hier stehen sämtlich Jar-Dateien - sowohl die eXTra-Jar-Dateien als auch die externen Bibliotheken zur Verfügung. Je nach
eingesetzter Datenbank muss in diesem Verzeichnis noch ein Datenbank-JDBC-Treiber abgelegt werden.

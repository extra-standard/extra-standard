eXTra apps

Hier befinden sich sämtliche Anwendungen von eXTra.

extra-cli
----------------------------------
eXTra CLI (Command Line Interface) ist eine Batch-Anwendung für eXTra, sowohl für
Client als auch Server.

extra-profiling
----------------------------------
eXTra Profiling Tool ist eine Anwendung (derzeit eine Swing-Anwendung) für 
die Generierung der eXTra-Profile.

extra-scenario-sendfetch
----------------------------------
eXTra Scenario ist eine Zusammenstellung von Konfigurationen, die für bestimmte Mandanten, Fachverfahren 
geeignet sind. Diese Auslieferung kann zusätzlich zum eXTra CLI deployed werden.

extra-webapp
----------------------------------
eXTra Webapp (Webanwendung) stellt eine Webanwendung sowohl für Client als auch Server 
für eXTra zur Verfügung.

extra-spoc
----------------------------------
eXTra Single Point of Contact (eXTra Server).

--------------------------------------
extra-profiling Version 1.0.0-SNAPSHOT
--------------------------------------
Unterstützung von eXTra-Standard Version 1.1


--------------------------------------
Verzeichnisbeschreibung
--------------------------------------
/bin
Hier stehen ausführbare Dateien zur Verfügung. Die Datei extra-profiling.bat kann aus dem Windows-Console gestartet werden. 
Die Umsetzung in Linux ist analog zu der extra-profiling.bat.

/conf
Hier stehen sämtliche Konfigurationsdateien zur Verfügung.

/lib
Hier stehen sämtlich Jar-Dateien - sowohl die eXTra-Profiling-Jar-Dateien als auch die externen Bibliotheken zur Verfügung.

/resource
Hier stehen sämtliche Informationen (XSD, Dokumentationen) zu der passenden eXTra-Spezifikation.
Die XSD-Dateien werden in extra-profiling verwendet.

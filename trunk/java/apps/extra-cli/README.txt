--------------------------------
extra-cli Version 1.0.0-SNAPSHOT
--------------------------------
Unterstützung von eXTra-Standard Version 1.3


--------------------------------
Verzeichnisbeschreibung
--------------------------------
/bin
Hier stehen ausführbare Dateien zur Verfügung. Die Datei extra-cli.bat kann aus dem Windows-Console gestartet werden. 
Die Umsetzung in Linux ist analog zu der extra-cli.bat.

/conf
Hier stehen sämtliche Konfigurationsdateien zur Verfügung.

/lib
Hier stehen sämtlich Jar-Dateien - sowohl die eXTra-Jar-Dateien als auch die externen Bibliotheken zur Verfügung.
